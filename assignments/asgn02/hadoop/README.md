INPUT: name, id, grade, course

ex)
John Back, 23, B, CSC366
Bob Wilson, 11, B, CS201
John Back, 23, A, CSC369

OUTPUT: name, id, (grade1, class1), (grade2, class2), ...

ex)
Bob Wilson, 11, (B, CS201)
John Back, 23, (A, CSC369), (B, CSC366) 

NOTE: the output is sorted by name, and then by grade
